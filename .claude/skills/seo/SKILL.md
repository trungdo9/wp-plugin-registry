# SEO & GEO Knowledge Base

Specialized knowledge for SEO (Search Engine Optimization) and GEO (Generative Engine Optimization).

## When to Use

Use this skill when working on:
- SEO audits and optimization
- Content optimization for search engines
- AI search visibility (ChatGPT, Claude, Perplexity)
- Core Web Vitals improvement
- E-E-A-T optimization
- Schema markup implementation

---

## Core Web Vitals Targets

| Metric | Good | Poor | How to Fix |
|--------|------|------|-----------|
| **LCP** (Largest Contentful Paint) | < 2.5s | > 4.0s | Optimize images, CDN, caching |
| **INP** (Interaction to Next Paint) | < 200ms | > 500ms | Reduce JavaScript, optimize event handlers |
| **CLS** (Cumulative Layout Shift) | < 0.1 | > 0.25 | Reserve space for images, ads |

## E-E-A-T Framework

### Experience (E)
- First-hand product/service usage
- Real-world examples and case studies
- Personal anecdotes and stories
- Customer testimonials and reviews

### Expertise (E)
- Author credentials and qualifications
- Technical certifications
- Years of experience
- Educational background
- Professional memberships

### Authoritativeness (A)
- Backlinks from reputable sources
- Media mentions and features
- Industry awards and recognition
- Expert endorsements
- Brand reputation

### Trustworthiness (T)
- HTTPS security
- Clear contact information
- Privacy policy
- Terms of service
- Transparent pricing
- Customer support channels

## Technical SEO Checklist

### On-Page SEO
- [ ] Title tag: 50-60 characters, primary keyword
- [ ] Meta description: 150-160 characters, call-to-action
- [ ] H1-H6 hierarchy: Logical structure
- [ ] URL structure: Clean, descriptive, keyword-rich
- [ ] Internal linking: Clear site structure
- [ ] Image alt text: Descriptive, keyword-rich
- [ ] Schema markup: JSON-LD for rich snippets

### Technical Foundation
- [ ] XML sitemap submitted to Google Search Console
- [ ] robots.txt configured correctly
- [ ] Canonical tags on all pages
- [ ] HTTPS enabled site-wide
- [ ] Mobile-friendly responsive design
- [ ] Core Web Vitals passing
- [ ] Structured data validated

### Performance
- [ ] Page speed < 3 seconds
- [ ] Images optimized and compressed
- [ ] CSS/JS minified
- [ ] Browser caching enabled
- [ ] CDN used for assets
- [ ] Lazy loading for below-fold content

## GEO (Generative Engine Optimization)

### What AI Cites

AI systems like ChatGPT, Claude, and Perplexity cite content based on:

| Element | Why AI Cites It |
|---------|-----------------|
| **Original Statistics** | Unique, verifiable data |
| **Expert Quotes** | Authority and credibility |
| **Clear Definitions** | Extractable, structured info |
| **Step-by-Step Guides** | Actionable, practical content |
| **Comparison Tables** | Organized, comparative data |
| **FAQ Sections** | Direct question-answer format |
| **Source Attribution** | Citable, traceable sources |

### GEO Best Practices

- [ ] FAQ sections with structured Q&A
- [ ] Author bylines with credentials
- [ ] Publication dates and "last updated"
- [ ] Statistics with clear sources
- [ ] Expert quotes attributed properly
- [ ] Clear, concise definitions
- [ ] Comprehensive glossary
- [ ] Original research and data
- [ ] Cited sources and references

## SEO vs GEO Comparison

| Aspect | Traditional SEO | GEO |
|--------|-----------------|-----|
| **Primary Goal** | Rank #1 in Google | Be cited in AI responses |
| **Platform** | Google, Bing | ChatGPT, Claude, Perplexity |
| **Key Metrics** | Rankings, CTR, traffic | Citation rate, AI mentions |
| **Focus Areas** | Keywords, backlinks | Entities, authority, freshness |
| **Content Style** | Keyword-optimized | Entity-rich, comprehensive |
| **Updates** | Periodic algo updates | Continuous learning |

## Common SEO Mistakes

### Content Mistakes
- ❌ Keyword stuffing
- ❌ Thin content
- ❌ Duplicate content
- ❌ Missing meta descriptions
- ❌ Poor heading structure

### Technical Mistakes
- ❌ Slow page speed
- ❌ Broken links (404 errors)
- ❌ Redirect chains
- ❌ Missing alt text
- ❌ Blocked in robots.txt

### UX Mistakes
- ❌ Intrusive interstitials
- ❌ Non-mobile-friendly design
- ❌ Hard-to-find content
- ❌ Poor site navigation
- ❌ Annoying pop-ups
